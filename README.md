# Projeto Books Api

## 💻 _Description_

Simples api para realizar CRUD de livros.

## 💡  Info

**A api contém 5 rotas, sendo elas:**

- createBook | POST("/")
- getBooks | GET("/")
- getBookById | GET("/{id}")
- updateBook | PUT("/{id}")
- deleteBook | DELETE("/{id}")

## 💻 Tecnologias/Pacotes utilizados

**Backend**

- Java (v17)
- Spring Boot (v3.0.2)
- Spring Boot DevTools (Dependencies)
- Lombok (Dependencies)
- Spring Web (Dependencies)
